🧱 Los cimientos

Tres archivos que sostienen todo lo demás.

Núcleo base
main.dartEl interruptor de encendido. Prende la app, la conecta con Supabase y decide si mostrarte el login o el inicio.

📖 Diccionario en cristiano

Las palabras raras que hemos usado, explicadas fácil.

Flutter / Dart

La herramienta y el idioma con que está hecha la app. Escribes una vez y sirve para Android, iPhone y web.

Supabase

Tu base de datos en la nube. La "fuente de verdad" que todos los celulares comparten.

RLS

El guardia que vive dentro de la base y revisa los permisos de cada quien, sin importar por dónde entren.

Trigger

Un ayudante automático de la base: cada vez que hay un movimiento, recalcula solo la existencia y el costo promedio.

RPC / Función

Una "orden con nombre" que la app le pide a la base (ej. "búscame elementos" o "dame el kardex").

API

El idioma con que la app y la base se hablan por internet.

Edge Function

Un programita que corre en la nube para tareas delicadas (como crear usuarios de forma segura).

Git / commit

El historial con "fotos" de tu código. Cada commit es una foto guardada; GitHub es donde vive el respaldo.

Offline-first

Que la app funcione primero en el teléfono y sincronice después, para no depender de la señal.
